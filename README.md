# react-native-faster

#### 项目介绍
基于react-native快速开发框架，封装了api层，config配置，mobx状态管理中心，react-navigation导航，通用Header等，采用了antd-mobile丰富的UI组件库，使用此框架开发者只需要关注页面和业务即可
